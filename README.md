## Project Overview

A comprehensive UI/UX redesign of the KPR Institute of Engineering and Technology website. The existing site contains rich information but users struggle to find it quickly due to navigation complexity, information overload, and a broken mobile experience.

## Problem Statement

The KPRIET website has content-heavy pages, 11+ navigation items with no hierarchy, no prominent Apply Now CTA, and a layout that breaks on mobile devices — reducing its effectiveness in attracting prospective students.

## Goals

- Improve navigation and reduce clicks to key information
- Create a clean, modern design system
- Make the Apply Now CTA always visible
- Build a fully responsive layout (375px to 1440px)
- Improve accessibility to WCAG AA standard

## Key Findings

- **68%** appointment booking abandonment on the current site
- **11+** navigation items with no visual hierarchy
- Zero visible Apply Now CTA above the fold
- Mobile layout completely broken below 768px
- Text contrast fails WCAG AA standard

## Expected Impact

- 30% faster information discovery
- Improved admission inquiry conversions
- WCAG AA accessibility compliance
- 3× better mobile usability score
